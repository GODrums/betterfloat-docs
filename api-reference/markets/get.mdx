---
title: "Redirect to Market"
description: "Redirect to a market's website."
openapi: "GET /redirect/{provider}/{name}"
authMethod: "none"
---

<Note>You can directly open these links in your browser</Note>

## How to use

```ts
const encodedName = encodeURIComponent(name);
// Open directly in browser
window.open(`http://api.betterfloat.com/redirect/${provider}/${encodedName}`);
// Open from a browser extension
chrome.tabs.create({ url: `http://api.betterfloat.com/redirect/${provider}/${encodedName}` });
```
